Proyecto de Análisis de Ventas en Power BI
Este repositorio contiene mi proyecto final de análisis de ventas desarrollado en Power BI, titulado "Tabera Jorge - Proyecto Final". El proyecto presenta un análisis detallado de ventas, costos, rentabilidad, comportamiento de clientes y desempeño de vendedores a través de múltiples visualizaciones y gráficos interactivos.

Contenido del Proyecto
PDF del Proyecto: Un documento titulado "Tabera Jorge - Proyecto Final" que incluye capturas de pantalla y descripciones detalladas de cada hoja del análisis en Power BI.
Archivos de Datos: Archivos de datos utilizados para la creación de las visualizaciones.
Informe de Power BI: El archivo .pbix del informe de Power BI para que puedas explorar el proyecto por ti mismo.
Puedes utilizar este repositorio para aprender más sobre la estructura del análisis de ventas, la utilización de Power BI para crear visualizaciones interactivas, y para explorar los datos y las técnicas utilizadas.
